# Webpack TypeScript Boilerplate

Basic webpack set up to compile TypesCript, SCSS and HTML

### Info
 - TypeScript
 - Scss
 - HTML

### Installation

Clone the repo, install the dependencies and devDependencies and start the watcher.

```sh
$ npm install
```
### Make a Build
```bash
npm run build:dev
```
### Development server and watch

```bash
npm run start
```
